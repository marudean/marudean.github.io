介紹

Lucent Sky AVM 在掃描失敗時會產生一個錯誤碼。這個錯誤碼可以幫助你分析發生錯誤的原因。在本文中，我們將幫助你找到你的錯誤碼，並建議你應該如何修正錯誤。

掃描錯誤碼列表

錯誤碼 描述 解決方式
-1000 內部錯誤導致應用程式無法被建置。 請聯絡技術支援。
-1011 找不到有效的應用程式。 請確認應用程式中包和了專案檔案或是「web.config」檔案。
-1021 應用程式無法被建置。 請確認應用程式可以被 Visual Studio 或是 MSBuild 建置。
-1031 應用程式為網站且先行編譯失敗。 請確認應用程式可以被 Visual Studio 或是 ASP.NET compiler 發佈。
-1032 先行編譯失敗，且「bin」目錄不存在。 請確認應用程式是否包含「bin」目錄。
-1091 至少一個在設定檔（例如「web.config」）中被參考的組件無法在應用程式的「bin」目錄中被找到。 請確認應用程式的「bin」目錄是否包含設定檔中參考的組件。
-1100 應用程式無法被建置。 請聯絡技術支援。
-1111 找不到有效的應用程式。 請確認應用程式中包和了「WEB-INF」目錄。
-1112 應用程式缺少程式碼根目錄。 請確認應用程式中包和了程式碼根目錄，通常是「java」、「src」或是「classes」。
-1113 應用程式的 「web.xml」檔案不存在。 請確認應用程式中含有「web.xml」檔案。
-1120 應用程式無法被建置。 請確認應用程式是否能夠建置。
-1121 應用程式無法被建置，因為 Java 或 Java 標籤檔案中的錯誤。 請確認應用程式是否能夠建置。
-1122 應用程式無法被建置，因為 Java 或 Java 標籤檔案中的錯誤，或是缺少 JAR 函式庫。 請確認應用程式是否能夠建置，以及所需要使用的 JAR 函式庫都已經存在。這個應用程式使用 Maven，使用 Maven 下載所需要的 JAR 函式庫並放置到「WEB-INF\lib」可能可以解決這個錯誤。
-1123 應用程式無法被建置，因為 JSP 檔案中的錯誤。 請確認應用程式是否能夠建置。
-1124 應用程式無法被建置，因為 JSP 檔案中的錯誤，或是缺少 JAR 函式庫。 請確認應用程式是否能夠建置，以及所需要使用的 JAR 函式庫都已經存在。這個應用程式使用 Maven，使用 Maven 下載所需要的 JAR 函式庫並放置到「WEB-INF\lib」可能可以解決這個錯誤。
-1125 應用程式無法被建置。 請確認應用程式是否能夠建置。
-1126 應用程式無法被建置，通常是因為無效的程式碼或是缺少 JAR 函式庫。 請確認應用程式是否能夠建置，以及所需要使用的 JAR 函式庫都已經存在。這個應用程式使用 Maven，使用 Maven 下載所需要的 JAR 函式庫並放置到「WEB-INF\lib」可能可以解決這個錯誤。
-1127 內部錯誤導致應用程式無法被建置。 請聯絡技術支援。
-1130 內部錯誤導致應用程式無法被建置。 請聯絡技術支援。
-1200 內部錯誤導致應用程式無法被建置。 請聯絡技術支援。
-1221 應用程式無法被建置。 請確認應用程式是否有錯誤的 PHP 語法。
