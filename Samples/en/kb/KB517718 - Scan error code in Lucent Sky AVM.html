Introduction

Lucent Sky AVM generates an error code when a scan failed. This error code will help you diagnose the cause of the failed scan. In this article, we will help you find your error code and suggest what you might try to correct the error.

More Information

Scan error codes list

Error code description Solution
-1000 The application could not be built due to an internal error. Contact technical support.
-1011 Could not locate a valid application. Check if the application has a project file or a "web.config" file.
-1021 The application could not be built. Check if the application can be built in Visual Studio or by MSBuild.
-1031 The application is a web site and precompilation failed. Check if the application can be published in Visual Studio or by the ASP.NET compiler.
-1032 Precompilation failed and the application is "bin" directory. Check if the application has a "bin" directory.
-1091 At least one assembly referenced in the configuration file (such as "web.config") was not found in the "bin" directory of the application. Check if the assemblies referenced in the configuration file is included in the "bin" directory of the application.
-1100 The application could not be built. Contact technical support.
-1111 Could not locate a valid application. Check if the application has a "WEB-INF" directory.
-1112 The application does not have a source root. Check if the application has a source root directory, commonly named "java", "src" or "classes".
-1113 The application doest not have a web.xml file. Check if the application has a "web.xml" file.
-1120 The application could not be built. Check if the application can be built.
-1121 The application could not be built due to errors in Java or Java tag file. Check if the application can be built.
-1122 The application could not be built due to errors in Java or Java tag files, or missing JAR libraries. Check if the application can be built, and has the required JAR libraries. This application uses Maven, so using Maven to download the required JAR libraries to "WEB-INF\lib" might solve this error.
-1123 The application could not be built due to errors in JSP files. Check if the application can be built.
-1124 The application could not be built due to errors in JSP files, or missing JAR libraries. Check if the application can be built, and has the required JAR libraries. This application uses Maven, so using Maven to download the required JAR libraries to "WEB-INF\lib" might solve this error.
-1125 The application could not be built. Check if the application can be built.
-1126 The application could not be built, commonly due to invalid source code or missing JAR libraries. Check if the application can be built, and has the required JAR libraries. This application uses Maven, so using Maven to download the required JAR libraries to "WEB-INF\lib" might solve this error.
-1127 The application could not be built due to an internal error. Contact technical support.
-1130 The application could not be built due to an internal error. Contact technical support.
-1200 The application could not be built due to an internal error. Contact technical support.
-1221 The application could not be built. Check if the application contain PHP syntax errors.
