Introduction

This article describes how to include Application Protection Library (APL) in a PHP application. Application Protection Library is an open-source enterprise security library that makes it easier for developers to make aplications both new and legacy more secure.

More Information

Application Protection Library for PHP is composed of an initiator PHP file that loads of PHP files. To include them in your application, it's recommended to reference the initiator file using a .htaccess file. This will automatically reference APL in all PHP files. Alternatively, you can also reference the initiator file individually in all PHP files that need to reference APL.

To use a .htaccess file to reference APL:

  1. Create a .htaaccess file under the root folder of the PHP application, or open the existing .htaccess file if there's already one.
  2. Add the following line to the .htaccess file:
    php_value auto_prepend_file "/APL/ApplicationProtectionLibrary.php"
  3. Save the .htaccess file.

To add rerefence to APL individually:

  1. Open a PHP file that need to reference APL, and insert the following at the first line:
    {% highlight php %} {% endhighlight %}
  2. Repeat step 1 with all PHP files that need to reference APL.
