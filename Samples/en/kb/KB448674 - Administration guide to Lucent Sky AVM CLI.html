Introduction

Lucent Sky AVM has several different UIs designed for developers, including CLI, Visual Studio Extension and Eclipse plug-in. Lucent Sky AVM CLI is a command line application that provides access to AVM Server in a console environment, allowing developers to integrate the identification and mitigation of security issues into existing software development or DevOps lifecycles. This guide convers the installation, configuration and uninstallation of Lucent Sky AVM CLI.

More Information

System Requirements

Processor, memory, hard disk space and operating system requirements:
Components Requirement
Hard Disk Space 40 MB
Memory Minimum: 1 GB (1.5 GB if running inside a virtual machine)
Recommended: 2 GB
Processor Minimum: 1.6 GHz processor
Recommended: 2.0 GHz processor

Lucent Sky AVM CLI can be installed on the following operating systems:

  • Windows Server 2008 R2 with SP1 and later
  • Windows 7 with SP1 and later

Lucent Sky AVM CLI can be installed on the following operating systems through Mono:

  • Mac OS X 10.3 and later
  • OpenSUSE 13.1 and later
  • Ubuntu 10.04 and later

Lucent Sky AVM CLI might work on other operating systems through Mono as well, but these scenarios are not supported.

Installing Lucent Sky AVM CLI

In this section, you will see:

  • A demonstration of installing Lucent Sky AVM CLI on your development computer and configure it to communicate with a Lucent Sky AVM server.

At the end of this section, you will be able to:

  • Install Lucent Sky AVM CLI on your development computer.
  • Configure Lucent Sky AVM CLI to communicate with a Lucent Sky AVM server.

To install and configure AVM CLI

  1. Double click on the AVM CLI setup file (CLEAR_CLI.exe). The file will automatically extract and create a CLEAR CLI folder at C:\Program Files\Lucent Sky\CLEAR CLI (or the location you chose during the extraction process).
  2. Navigate to C:\Program Files\Lucent Sky\CLEAR CLI (or the location you chose during the extraction process). Open SkyAnalyzer.Interface.Console.exe with a text editor.
  3. Replace all occurrences of clear.contoso.com with the FQDN, IP address or computer name of a Lucent Sky AVM server, and then save and close the file.

In the default configuration, the communication between Lucent Sky AVM server and Lucent Sky AVM CLI is not secured. Consult with your system administrator about how to enable transport and/or message security on WCF TCP endpoints.

Uninstalling Lucent Sky AVM CLI

To uninstall AVM Visual Studio Extension

  1. Remove the directory C:\Program Files\Lucent Sky\CLEAR CLI (or the location you chose during the extraction process).
